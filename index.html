KIB in Batch Home

KIB in Batch is a lightweight Unix-like environment for Microsoft Windows.

Installation

We recommend you get the latest release here.

Features

  • Built-in package manager
  • Blazingly fast
  • Very few dependencies
  • Uninstaller can back up home directory, which is automatically restored if you reinstall

KIB in Batch 11

Installation

If you for some reason want version 11 (still in early development), here's how to get it.

Run these in a PowerShell session: 


			git clone https://github.com/KIB-in-Batch/kib-in-batch
			Set-Location -Path "kib-in-batch"
			src\kib_in_batch.bat

You can then test edge cases, hack around with it (a lot of the code is batch files), and submit bug reports.

Roadmap

  • More advanced package manager features
  • More complete and faster POSIX layer
  • Modular system (the software itself as a package)
