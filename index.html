KIB in Batch

The KIB in batch project aims to create a lightweight POSIX environment for Microsoft Windows.

It includes headers to compile POSIX applications with MSVC, a root filesystem and Win32 BusyBox programs.

Currently, the stable version is 10.2.6.

Installation

For Most People

You should get the latest stable release.

For Testers and Developers

First, clone the repository https://codeberg.org/KIB-in-Batch/installer.

Then, in your terminal, navigate to it and run src\kib_in_batch.bat.
